SpringBoot API Endpoints

/home

Welcome message

To reach this endpoint, visit: https://vedant-springrest.onrender.com/home

GET /home

/courses

Get all courses

To reach this endpoint, visit: https://vedant-springrest.onrender.com/courses

GET /courses

/courses/{courseId}

Get a specific course by ID

Replace `{courseId}` with an actual course ID. Example: https://vedant-springrest.onrender.com/courses/1

GET /courses/{courseId}

/courses (POST)

Add a new course

To reach this endpoint, use an API client like Postman or cURL with a POST request to: https://vedant-springrest.onrender.com/courses

POST /courses

/courses (PUT)

Update a course

To reach this endpoint, use an API client like Postman or cURL with a PUT request to: https://vedant-springrest.onrender.com/courses

PUT /courses

/courses/{courseId} (DELETE)

Delete a course by ID

Replace `{courseId}` with an actual course ID. Example: https://vedant-springrest.onrender.com/courses/1

To reach this endpoint, use an API client like Postman or cURL with a DELETE request to: https://vedant-springrest.onrender.com/courses/1

DELETE /courses/{courseId}
